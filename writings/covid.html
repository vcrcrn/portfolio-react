def bootstrap_sample(data, k):
    """
    Performs bootstrap sampling on data to obtain k samples of size n.
    
    Arguments:
        data - Dataset contained as a Pandas DataFrame 
        k - Number of randomly drawn samples
    
    Returns:
        samples - List containing k Pandas DataFrames of size n each
                  corresponding to each sample  
    """
    samples = []
    for x in range(k):
        samples.append(data.sample(n=len(data), replace=True))
    return samples

def extract_coefs(models, include_intercept = True):
    """
    NOTE: This function has already been implemented. You do not need to modify this!
    
    Extracts coefficients of all the linear regression models in MODELS, and returns
    it as a NumPy array with one model's coefficients as each row.
    
    Arguments:
        models - Contains k sklearn LinearRegression models, each with p + 1 coefficients
        include_intercept - Whether to include intercept in returned coefficients
    
    Returns:
        coef_array - Coefficients of all k models, each with p + 1 coefficients (if intercept
                     enabled, otherwise p). Returned object is k x (p + 1) NumPy array.
    """
    coef_array = np.zeros(shape = (len(models), len(models[0].coef_) + 1))
    for i, m in enumerate(models):
        coef_array[i, 0] = m.intercept_
        coef_array[i, 1:] = m.coef_
    if include_intercept:
        return coef_array 
    return coef_array[:, 1:]

def confidence_interval(coefs):
    """

    bootstrapped models. Returns output as a list of confidence intervals.
    
    Arguments:

                coefficients of bootstrapped models
    
    Returns:
        cis - Confidence intervals of each parameter theta_i in the form of a 
              list like this: [(0.5, 0.75), (0.2, 0.4), ...]
    """
    cis = []
    
    # FILL IN CODE BELOW
    for i in range(coefs.shape[1]):
        theta_i_values = coefs[:, i]
        theta_i_lower_ci, theta_i_upper_ci = np.percentile(theta_i_values, 2.5), np.percentile(theta_i_values, 97.5)

Interpretation¶

If the variables are related to each other, then they will be collinear. This means that a feature can be predicted pretty accurately by a linear combination of the other features. This is why the confidence intervals were so small. The model was not very accurate because it was overfitting the data.



Performing Bias-Variance Analysis¶

In this part I performed an analysis that could estimate the bias and variance of our models, which could be helpful in modeling.

I used the same bootstrapped models contained in the Python list models to estimate our model variance. To do this, I remembered that the model variance on a data point is simply the variance of the predictions on that sample point. From the bias-variance decomposition, for a parametric model $\hat{Y}(x) = f_{\hat{\theta}}(x)$:

$$\text{model variance} = \mathrm{Var}(f_{\hat{\theta}}(x))$$

To investigate the variance in the test predictions, I sampled a particular data point $(x_i, y_i)$. Defining the model risk for this point as the mean square error over all possible fitted models:

$$\text{model risk} = \mathbb{E}\left[\left(y_i - f_{\hat{\theta}}(x_i)\right)^2\right]$$

Using the bootstrapped estimates models, I approximated the ratio of model variance to model risk for the datapoint $i = 100$, i.e., $(x_{100}, y_{100})$. I could interpret this ratio as the the proportion of the expected square error on the data point "captured" by the model variance. Since I used the bootstrap to create 1000 models, I generated 1000 predictions for the given $x_i$. Where X was the design matrix of mask features, and y was the 9/12/2021_cpc response.

I then assigned prop_var to the computed, approximate ratio:

$$ \frac{\text{model variacne}}{\text{model risk}}=\frac{\mathrm{Var}\left(f_{\hat{\theta}}(x_{100})\right)}{\mathbb{E}\left[\left(y_{100} - f_{\hat{\theta}}(x_{100})\right)^2\right]} $$

The model variance is not the dominant term in the bias-variance decomposition. The model bias is the dominant term.

$$ \text{model risk} = \sigma^2 + (\text{model bias})^2 + \text{model variance} $$

where $\sigma^2$ was the observation variance, or "irreducible error".

Then, I calculated the average variance and average mean square error across 250 randomly sampled $(x_i, y_i)$ points. In other words, I estimated the following quantities across all $x_i$ and $y_i$ in X_sample and y_sample:

$$ \frac{1}{250} \sum_{i=1}^{250} \mathrm{Var}\left(f_{\hat{\theta}}(x_i)\right) $$

and

$$ \frac{1}{250} \sum_{i=1}^{250} \mathbb{E}\left[\left(y_i - f_{\hat{\theta}}(x_i)\right)^2\right] $$
